## Default Permission

Default permissions for the TTS plugin - allows all text-to-speech operations

#### This default permission set includes the following:

- `allow-speak`
- `allow-stop`
- `allow-get-voices`
- `allow-is-speaking`
- `allow-pause-speaking`
- `allow-resume-speaking`
- `allow-preview-voice`
